## Laravel-Vue-Zhihu
使用 Laravel 开发知乎

## 主要功能
- 用户注册
- 邮件通知
- 发布问题
- 回答问题
- 用户关注
- 用户私信

采用 PHP，MySQL，Vue.js, gulp 等技术。
Laravel 框架创建项目后，设计用户表、问题表、回答表、评论表，使用 RESTful 资源路由实现问题的 CRUD 操作。
前端使用 blade 模板及 VueJS 组件化开发。

邮件服务使用[Laravel-SendCloud](https://github.com/NauxLiu/Laravel-SendCloud)
